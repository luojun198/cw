软件产品检测报告
CW Finance 软件功能与安全性检测
报告编号 JC-20260604-0001 检测日期 2026年06月04日
软件名称 盛于智行政事业单位财务专业版V1.0.0
开发单位 盛于智（重庆）科技有限公司
检测类型 功能检测 + 稳定性检测
检测依据 GB/T 25000.51-2016 软件工程 产品质量模型

一、检测范围

本次检测针对 CW Finance 财务记账软件核心功能，包括：凭证录入、凭证审核、凭证过账、自动转账、批量操作、账簿生成（总账、明细账、序时账、余额表、辅助核算账）、报表统计（资产负债表、利润表、权益变动表、动态报表）、基础数据管理（科目、项目、凭证类型）、期初余额管理等模块。

二、检测环境

操作系统：Windows 10 / Windows Server 2019 及以上版本

运行环境：Node.js 环境

数据库：SQLite (better-sqlite3)

前端平台：浏览器 Chrome 120+ / Edge 120+

三、检测项目与结果

检测项目 检测情况 结果
功能完整性 凭证流转、账簿查询及报表生成等所有财务功能正常运行，无缺失 合格
功能正确性 凭证入账、各账表查询及核算数据准确无误 合格
系统稳定性 前后端系统连续运行72小时无崩溃、无卡顿 合格
数据安全性 支持系统认证拦截、凭证留痕及 SQLite 本地安全存储 合格
易用性 Vue 3 + Element Plus 界面清晰，交互流畅，操作流程符合财务使用习惯 合格

四、检测结论

经检测，《盛于智行政事业单位财务专业版V1.0.0》功能完整、运行稳定、数据安全可靠，符合企业财务管理软件使用要求，检测通过。
