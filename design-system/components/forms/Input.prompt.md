Labelled text field — the workhorse form input, with a soft mint focus ring and optional hint/error.

```jsx
<Input label="邮箱" placeholder="you@example.com" hint="用于登录与通知" />
<Input label="金额" prefix="¥" error="请输入有效金额" />
```

Props: `label`, `hint`, `error`, `prefix`, `suffix`, `size` (`sm | md | lg`), plus native input props.
