# 前端UI彻底改造计划

## 设计理念

**Mac 风格 × 中国政府财务系统 × 严肃科技感**

| 维度 | 关键词 | 说明 |
|------|--------|------|
| 🍎 **Mac风格** | 简洁、通透、大留白、毛玻璃、SF字体体系 | 借鉴 macOS 的视觉语言，但不失严肃 |
| 🏛️ **政府财务** | 庄重、规范、可信赖、数据精准 | 深蓝/藏青为主色，金色为点缀 |
| 🔬 **科技感** | 微动效、渐变、光晕、数据可视化 | subtle animations, glassmorphism, 数据仪表盘 |

## 六、交互动效体系

### 6.1 过渡动画

```css
/* 页面切换 */
.page-enter { opacity: 0; transform: translateY(8px) }
.page-enter-active { transition: all 0.25s cubic-bezier(0.4, 0, 0.2, 1) }

/* 对话框弹出 */
.dialog-enter-from { opacity: 0; transform: scale(0.95) }
.dialog-enter-active { transition: all 0.2s cubic-bezier(0.4, 0, 0.2, 1) }

/* 按钮悬停 */
.button-hover { transform: translateY(-1px); box-shadow: 0 4px 12px rgba(...) }
```

### 6.2 微交互

| 场景 | 动效 | 时长 |

## 十、技术实现要点

### 10.1 Element Plus 全局覆写

```scss
// 不修改 Element Plus 源码，在 styles.css 中用 CSS 变量 + 选择器覆写
// 通过 :root 变量 + [data-theme] 实现亮暗切换

// 表格无边框
.el-table--borderless {
  border: none;
  .el-table__inner-wrapper::before { display: none }
}
```

### 10.2 毛玻璃兼容

```css
// Firefox 不支持 backdrop-filter，使用半透明背景兜底
.glass {
  background: rgba(255, 255, 255, 0.85);
  @supports (backdrop-filter: blur(20px)) {
    background: rgba(255, 255, 255, 0.6);
    backdrop-filter: blur(20px);
  }
}
```

### 10.3 字体加载策略

- PingFang SC 在 macOS 上原生自带
- Windows 回退到 Microsoft YaHei
- 数字统一使用 SF Mono / JetBrains Mono
