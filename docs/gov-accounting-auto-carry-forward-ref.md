# 中国行政事业单位会计准则——自动结转文献综述

> 整理日期：2026年4月12日
> 来源：互联网公开文献及财政部官方文件

## 一、法规与制度依据

### 1.1 《政府会计准则——基本准则》（财政部令第78号）

- 发布时间：2015年10月23日
- 核心内容：确立了政府会计"双功能、双基础、双报告"的核算模式
  - 财务会计实行权责发生制
  - 预算会计实行收付实现制
- 来源：<https://www.moj.gov.cn/pub/sfbgw/flfggz/flfggzbmgz/201511/t20151125_145702.html>

### 1.2 《政府会计制度——行政事业单位会计科目和报表》（财会〔2017〕25号）

- 发布时间：2017年10月24日
- 施行时间：2019年1月1日
- 核心内容：
  - 规定了行政事业单位会计科目设置及使用说明
  - 明确了期末结转的科目和流程
  - 设置了"累计盈余"科目，涵盖原制度下"财政拨款结转""财政拨款结余""其他资金结转结余"的余额内容
- 来源：<http://kjs.mof.gov.cn/zhengcefabu/201711/t20171109_2746877.htm>

### 1.3 《中央部门结转和结余资金管理办法》（财预〔2016〕18号）

- 核心内容：
  - **结转资金**：预算未全部执行或未执行，下年需按原用途继续使用的预算资金
  - **结余资金**：项目实施周期已结束、项目目标已完成或项目提前终止，尚未列支的项目支出预算资金
  - 基本支出结转：年度预算执行结束时，尚未列支的基本支出全部作为结转资金管理
  - 项目支出结余：原则上由财政部收回
  - 清理期限：年度预算执行结束后45日内完成清理
- 来源：<http://www.mof.gov.cn/gkml/caizhengwengao/wg2016/wg201604/201608/t20160812_2387096.htm>

### 1.4 《会计信息化工作规范》（财会〔2024〕11号）

- 施行时间：2025年1月1日
- 核心内容：
  - 会计信息系统自动生成且具有明确审核规则的会计凭证，可将审核规则嵌入系统进行自动审核
  - 系统自动审核的规则应当可查询、可校验、可追溯
  - 会计软件应提供按规定会计期间进行结账的功能
  - 结账前应自动检查本期凭证是否全部登记入账
  - 会计软件应提供不可逆的记账功能
- 来源：<https://kjs.mof.gov.cn/zhengcefabu/202408/P020240805628932632907.pdf>

### 1.5 《会计软件基本功能和服务规范》（财会〔2024〕12号）

- 核心内容：
  - 会计软件应支持审核程序自动化
  - 能够根据预设审核规则实现凭证数据、业务数据和资金支付数据的自动关联和相互校验
  - 校验无误的电子原始凭证可自动填制电子记账凭证
- 来源：财政部会计司官网

### 1.6 政府会计准则制度解释（第1号至第6号）

- **解释第1号**：涉及期末结转的基本流程和科目对应关系
- **解释第3号**：细化了结转结余的会计差错更正处理
- **解释第5号**：补充了财政拨款结转结余的特殊情形处理
- **解释第6号**：进一步明确了结转结余相关业务的会计处理
- 来源：
  - <https://zcpgzx.mof.gov.cn/zcfg/202103/t20210304_3665858.htm>
  - <https://www.mof.gov.cn/jrttts/202311/t20231101_3914252.htm>

### 1.7 《关于行政事业单位划转撤并有关会计处理规定》（财会〔2022〕23号）

- 核心内容：
  - 清算期间增设"清算收入""清算费用""清算净损益"科目
  - 清算结束日将"清算收入"和"清算费用"转入"清算净损益"
  - 将"清算净损益"转入"累计盈余"
  - 结账后除"累计盈余""专用基金""权益法调整"外，其他净资产类科目应无余额
- 来源：<http://kjs.mof.gov.cn/gongzuotongzhi/202207/P020220725321137090151.pdf>

## 二、期末结转的核心流程

### 2.1 财务会计期末结转

政府会计实行"双功能"核算，财务会计期末结转主要包括以下步骤：

#### 第一步：结转损益类科目至"本期盈余"（科目3301）

**结转收入类科目：**

| 借方科目 | 贷方科目 |
|---------|---------|
| 财政拨款收入 | 本期盈余 |
| 事业收入 | 本期盈余 |
| 上级补助收入 | 本期盈余 |
| 附属单位上缴收入 | 本期盈余 |
| 经营收入 | 本期盈余 |
| 非同级财政拨款收入 | 本期盈余 |
| 投资收益 | 本期盈余 |
| 捐赠收入 | 本期盈余 |
| 利息收入 | 本期盈余 |
| 租金收入 | 本期盈余 |
| 其他收入 | 本期盈余 |

**结转费用类科目：**

| 借方科目 | 贷方科目 |
|---------|---------|
| 本期盈余 | 业务活动费用 |
| 本期盈余 | 单位管理费用 |
| 本期盈余 | 经营费用 |
| 本期盈余 | 资产处置费用 |
| 本期盈余 | 上缴上级费用 |
| 本期盈余 | 对附属单位补助费用 |
| 本期盈余 | 所得税费用 |

#### 第二步：年末结转"本期盈余"至"本年盈余分配"

- 若为盈余：借：本期盈余 / 贷：本年盈余分配
- 若为亏损：借：本年盈余分配 / 贷：本期盈余

#### 第三步：将"本年盈余分配"结转至"累计盈余"

- 借：本年盈余分配 / 贷：累计盈余（或作相反分录）
- 结转后"本年盈余分配"科目应无余额

### 2.2 预算会计期末结转

#### 财政拨款结转（科目8101）

1. **收支结转**：
   - 将"财政拨款预算收入"本年发生额转入"8101 财政拨款结转——本年收支结转"
   - 将各项支出中属于财政拨款支出的本年发生额转入"8101 财政拨款结转——本年收支结转"

2. **冲销明细**：
   - 将"8101 财政拨款结转"下除"累计结转"以外的明细科目余额转入"累计结转"明细科目
   - 年末结账后，除"累计结转"明细科目外，其他明细科目应无余额

3. **结余转入**：
   - 分析各项财政拨款结转项目，将符合财政拨款结余性质的项目余额转入"财政拨款结余——结转转入"

#### 非财政拨款结转（科目8201）

- 核算本年度非同级财政拨款专项收支相抵后的余额
- 年终结账后，该科目明细科目应无余额

#### 非财政拨款结余（科目8301）

- 非财政拨款结余资金在扣除相关项目后，可按照规定进行结余分配（如提取专用基金等）

## 三、自动结转的实现机制

### 3.1 自动结转的定义

自动结转是指根据用户定义的规则，由会计软件自动生成转账凭证的方法。这是电算化会计系统中非常成熟且常用的功能，能够有效减少手工操作的繁琐与差错。

### 3.2 自动结转的设置与执行

#### 系统初始化阶段

- 财务人员需要定义自动转账分录（如结转损益、计提折旧、费用分摊等）
- 设置结转的科目范围、结转方式（如按比例、按余额等）
- 配置是否自动生成凭证

#### 期末处理阶段

1. 登录系统，进入"结账"或"期末处理"菜单
2. 选择需要进行期末结转的会计期间
3. 点击"测算金额"，系统自动计算各结转项目的金额
4. 在"结转损益"模板下点击"生成凭证"
5. 系统根据预设公式自动获取数据、生成凭证并完成结转

#### 凭证生成形式

- **合并结转**：将所有收入和费用合并生成一张凭证
- **分开结转**：分别生成收入结转凭证和费用结转凭证

### 3.3 自动结转的关键技术要求

根据《会计信息化工作规范》（财会〔2024〕11号）和《会计软件基本功能和服务规范》（财会〔2024〕12号）：

1. **自动审核**：具有明确审核规则的会计凭证，可将审核规则嵌入系统进行自动审核
2. **可追溯性**：系统自动审核的规则应当可查询、可校验、可追溯
3. **审批程序**：自动审核规则的设定与变更应当履行审批程序，严格管理，并留档备查
4. **内部控制**：将控制流程和控制规则嵌入会计信息系统，实现对违反控制要求情况的自动防范和监控预警
5. **不可逆记账**：会计软件应提供不可逆的记账功能，不得提供对已记账凭证的删除和插入功能
6. **结账检查**：结账前应自动检查本期输入的会计凭证是否全部登记入账

### 3.4 政府会计制度下的自动结转特点

政府会计实行"双功能"核算，自动结转需同时处理：

| 维度 | 财务会计 | 预算会计 |
|------|---------|---------|
| 核算基础 | 权责发生制 | 收付实现制 |
| 结转目标 | 本期盈余→累计盈余 | 财政拨款结转/非财政拨款结转/结余 |
| 结转频率 | 月末/季末/年末 | 年末 |
| 结转后状态 | 损益类科目余额为零 | 预算收支类科目余额为零 |

## 四、结转结余资金管理

### 4.1 结转与结余的区分

| 类别 | 定义 | 处理方式 |
|------|------|---------|
| 基本支出结转 | 年度预算执行结束时，尚未列支的基本支出 | 全部作为结转资金管理，结转下年继续用于基本支出 |
| 项目支出结转 | 已批复的项目预算尚未列支的部分（除连续两年未用完的外） | 作为结转资金管理，结转下年按原用途继续使用 |
| 项目支出结余 | 项目实施周期已结束或提前终止，尚未列支的项目支出预算 | 原则上由财政部收回 |

### 4.2 结余资金的清理与收回

- **清理期限**：年度预算执行结束后，中央部门应在45日内完成对结余资金的清理
- **收回程序**：财政部收到清理情况后，应在30日内发文收回结余资金
- **决算差异处理**：决算中项目支出结余资金数与已收回数不一致的，财政部根据批复的决算及时调整

### 4.3 预算调整与控制

- 中央部门拟调减预算或对结转资金用途进行调整，应按规定程序在8月31日前提出申请
- 对于结转资金规模较大、占年度支出比例较高的，财政部可收回部分结转资金

## 五、会计差错更正与调整

### 5.1 以前年度差错更正

- 涉及以前年度支出收回或会计差错更正的，通过"以前年度盈余调整"科目核算
- 涉及预算收支调整的事项，通过结转结余科目下的"年初余额调整"明细科目进行核算

### 5.2 报告日后调整事项

- 对于报告日后发生的调整事项，单位应按规定在发现差错的期间进行账务处理
- 调整相关会计报表及附注的金额
