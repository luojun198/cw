# 项目概述

## 项目名称

纯 Web 版行政事业单位财务记账系统

## 项目描述

面向行政事业单位的纯 Web 财务记账软件，基于政府会计制度，支持凭证管理、账簿查询、报表生成、AI 辅助等功能，支持内网部署。

## 技术栈

- **前端**: Vue 3 + Vite + TypeScript + Element Plus + Pinia + Vue Router + Axios
- **后端**: Node.js + Express.js + better-sqlite3 (SQLite WAL 模式)
- **认证**: JWT (8h 有效期) + RBAC 权限控制
- **AI**: OpenAI API 集成（智能摘要、异常检测）
- **数据库**: SQLite，14 张表，64MB 缓存，WAL 模式
- **开发**: tsx 运行时（ESM 模式），monorepo 结构

## 目录结构

```

├── client/                    # Vue3 前端
│   ├── src/
│   │   ├── api/               # API 请求封装 (request.ts, auth.ts)

│   │   ├── views/             # 页面视图
│   │   │   ├── system/        # 账套、用户、角色、参数、日志
│   │   │   ├── base/          # 科目、凭证类型、项目、期初余额
│   │   │   ├── voucher/       # 凭证录入、审核、查询
│   │   │   ├── ledger/        # 总账、明细账、余额表、日记账
│   │   │   ├── aux/           # 部门、项目、往来单位等
│   │   │   ├── report/        # 资产负债表、收入费用表、现金流量表、净资产变动表、财政拨款收支表
│   │   │   └── security/      # 备份恢复
│   │   ├── router/            # 路由配置
│   │   └── assets/            # 静态资源
│   └── dist/                  # 构建产物
├── server/                    # Node.js 后端
│   └── src/
│       ├── db/                # 数据库 (schema.sql, index.ts, seedAccounts.ts)
│       ├── middleware/         # 认证 (auth.ts)、操作日志 (log.ts)
│       ├── routes/            # API 路由
│       └── scripts/           # 初始化脚本
└── data/                      # SQLite 数据库文件目录
```

### 报表 API

- `GET /api/report/balance-sheet?year=&period=` → 资产负债表（资产净值=原值-累计折旧）
- `GET /api/report/income-statement?year=&period=` → 收入费用表（本期盈余=收入-费用）
- `GET /api/report/cash-flow?year=&period=` → 现金流量表（经营/投资/筹资三类）
- `GET /api/report/equity-changes?year=&period=` → 净资产变动表（期初/增加/减少/期末）
- `GET /api/report/fiscal-appropriation?year=&period=` → 财政拨款收入支出表
- `GET /api/report/aux-balance?year=&period=&aux_type=` → 辅助余额表
- `POST /api/report/ai/anomaly-check` → AI凭证异常检测
- `GET /api/report/ai/config` → AI配置查询
- `PUT /api/report/ai/config` → AI配置保存

### Dashboard API

- `GET /api/dashboard/stats` → 工作台统计数据（本月凭证数/借方合计/贷方合计/待审核数）
- `GET /api/dashboard/recent-vouchers` → 近期凭证（最近10条）
