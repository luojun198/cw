# CW Finance 快捷键说明

本文档记录系统中所有已注册的快捷键，新增快捷键前请检查是否冲突。

## 全局快捷键

| 快捷键 | 功能 | 使用场景 | 文件位置 |
|--------|------|----------|----------|
| Ctrl+S | 保存 | 凭证录入表单、核算项目弹窗 | `client/src/components/voucher/VoucherEntryForm.vue`<br>`client/src/views/base/Project.vue` |
| Enter | 保存 | 凭证录入表单（摘要联想框内除外）、核算项目弹窗（备注框内为换行） | `client/src/components/voucher/VoucherEntryForm.vue`<br>`client/src/views/base/Project.vue` |
| Ctrl+Enter | 保存并新增 | 凭证录入表单（含摘要框内）、核算项目新增弹窗 | `client/src/components/voucher/VoucherEntryForm.vue`<br>`client/src/views/base/Project.vue` |
| Ctrl+R | 刷新 | 凭证录入页面（刷新草稿列表） | `client/src/views/voucher/Entry.vue` |
| Ctrl+F | 文件选择 | 凭证录入表单（上传附件） | `client/src/components/voucher/VoucherEntryForm.vue` |
| Ctrl+E | 批量操作 | 凭证审核页面（打开批量操作对话框） | `client/src/views/voucher/Audit.vue` |
| Ctrl+D | 批量反记账 | 凭证审核页面 | `client/src/views/voucher/Audit.vue` |
| Escape | 关闭 | 凭证录入表单 | `client/src/components/voucher/VoucherEntryForm.vue` |
| Delete | 删除 | （预留，未使用） | `client/src/composables/useKeyboardShortcuts.ts` |
| Ctrl+Shift+L | 品牌设置 | 全局：跳转系统参数页并展开隐藏的品牌配置区 | `client/src/views/Layout.vue` |

## 快捷键冲突检查

### 已占用的组合键摘要
- ✅ Ctrl+S - 保存
- ✅ Enter - 保存（凭证表单、核算项目弹窗）
- ✅ Ctrl+Enter - 保存并新增（凭证表单、核算项目弹窗）
- ✅ Ctrl+R - 刷新
- ✅ Ctrl+F - 文件选择
- ✅ Ctrl+E - 批量操作
- ✅ Ctrl+D - 批量反记账
- ✅ Ctrl+Shift+L - 品牌设置（隐藏页）
- ❌ Ctrl+L - **浏览器占用**（聚焦地址栏），不可用

### 可用的 Ctrl 组合键（建议）
- Ctrl+P - 打印
- Ctrl+Q - 退出
- Ctrl+A - 全选
- Ctrl+Z - 撤销
- Ctrl+Y - 重做
- Ctrl+X - 剪切
- Ctrl+C - 复制
- Ctrl+V - 粘贴
- Ctrl+W - 可用（注意浏览器冲突）

## 快捷键实现说明

快捷键通过 `useKeyboardShortcuts` composable 统一管理，位于：
- `client/src/composables/useKeyboardShortcuts.ts`

### 使用方法

```typescript
import { useKeyboardShortcuts } from '@/composables/useKeyboardShortcuts'

// 注册快捷键
useKeyboardShortcuts([
  { key: 's', ctrl: true, handler: handleSave, description: 'Ctrl+S 保存' },
  { key: 'Escape', handler: handleClose, description: 'Esc 关闭' }
])
```

### 注意事项

1. **检查冲突**：新增快捷键前先查看本文档，避免与已有快捷键冲突
2. **更新文档**：注册新快捷键后必须更新本文档
3. **用户提示**：重要快捷键应在界面上提示用户（如按钮 title 属性）
4. **浏览器冲突**：避免使用浏览器默认快捷键。常见由浏览器占用、页面通常**无法**用 `preventDefault` 抢到的组合包括：**Ctrl+N**（新窗口）、**Ctrl+T**（新标签）、**Ctrl+W**（关标签）、**Ctrl+Shift+N**（Chrome 无痕）等。凭证/核算项目保存：**Enter** 保存、**Ctrl+Enter** 保存并新增；摘要联想框内 **Enter** 不保存（请用 **Ctrl+Enter** 或按钮）。
5. **跨平台**：考虑 Mac 用户（Cmd 键在代码中与 Ctrl 一并匹配）
