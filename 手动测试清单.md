## 六、数据一致性测试

### 6.1 借贷平衡测试

**测试目的：** 验证所有凭证借贷平衡

**验证方法：**
```sql
-- 检查所有凭证的借贷平衡
SELECT v.id, v.voucher_no, v.voucher_date,
       SUM(CASE WHEN ve.direction = 'debit' THEN ve.amount ELSE 0 END) as total_debit,
       SUM(CASE WHEN ve.direction = 'credit' THEN ve.amount ELSE 0 END) as total_credit,
       ABS(SUM(CASE WHEN ve.direction = 'debit' THEN ve.amount ELSE 0 END) -
           SUM(CASE WHEN ve.direction = 'credit' THEN ve.amount ELSE 0 END)) as diff
FROM vouchers v
JOIN voucher_entries ve ON ve.voucher_id = v.id
WHERE v.account_set_id = (SELECT id FROM account_sets LIMIT 1)
GROUP BY v.id, v.voucher_no, v.voucher_date
HAVING diff > 0.01
ORDER BY v.voucher_date DESC;
```

**预期结果：**
- ✅ 返回 0 行（所有凭证借贷平衡）

---

### 6.2 科目余额一致性测试

**测试目的：** 验证科目余额与凭证分录一致

**验证方法：**
```sql
-- 对比 account_balances 和 voucher_entries 的金额
SELECT
  ab.account_code,
  ab.current_debit as balance_debit,
  ab.current_credit as balance_credit,
  COALESCE(SUM(CASE WHEN ve.direction = 'debit' THEN ve.amount ELSE 0 END), 0) as entry_debit,
  COALESCE(SUM(CASE WHEN ve.direction = 'credit' THEN ve.amount ELSE 0 END), 0) as entry_credit
FROM account_balances ab
LEFT JOIN voucher_entries ve ON ve.account_id = ab.account_id
LEFT JOIN vouchers v ON v.id = ve.voucher_id
  AND v.year = ab.year
  AND v.period = ab.period
  AND v.status = 'posted'
WHERE ab.account_set_id = (SELECT id FROM account_sets LIMIT 1)
  AND ab.year = 2026
  AND ab.period = 1
GROUP BY ab.account_code, ab.current_debit, ab.current_credit
HAVING ABS(balance_debit - entry_debit) > 0.01
    OR ABS(balance_credit - entry_credit) > 0.01;
```

**预期结果：**
- ✅ 返回 0 行（余额表与分录一致）

---

### 6.3 整数字段一致性测试

**测试目的：** 验证 amount 和 amount_cents 一致

**验证方法：**
```sql
-- 检查 amount 和 amount_cents 的一致性
SELECT id, amount, amount_cents,
```
